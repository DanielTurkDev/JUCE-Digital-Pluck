Description:
Project inlcudes an efficent real-time buffer based exponential decay "plucked" sine wave plugin.
Plugin includes a UI allowing for familiar interaction with the sound generator within a DAW or as a standalone plugin.
Project must be built using ProJuicer.

Script based Audio Processor prototype built using Matlab included to demonstrate earlier stages in the development process.
